# SQL Server and Visual Studio

When developing locally in Visual Studio, you can use a built-in light version of SQL Server called `localdb`.

## Creating a new Database / Query
1. Open **SQL Server Object Explorer**.
2. Navigate to: `(localdb)\MSSQLLocalDB` $\rightarrow$ `Databases` $\rightarrow$ `System Databases` $\rightarrow$ `master`.
3. Right click $\rightarrow$ `New Query`.

## Useful Visual Studio Shortcuts
- `Ctrl + ;` : Open Search in the Solution Explorer.
- `Ctrl + \`, `Ctrl + S` : Open the SQL Server Object Explorer.
